你是一个理性的人还是一个感性的人？ 我从未觉得自己是一个理性的人，也从未觉得自己一直是感性的人。因为我觉得自己是游荡在理性和感性之间的 “小丑”，口头上的理性内心的感性 。 比如当 “ 前途重要还是感情重要 ” 这样一个经典的问题抛给我时，我会理性到极端毫不犹豫地回答前途重要，并且还会扯出一堆似乎很在理的理由。但是真正面临这种事情的时候，我的理性和感性就会开始一场漫长的斗争，最后往往会是理性完败。 是的，当事情真正发生时，我的理性总是很难战胜感性，明知是错误的选择但还是会选择，正所谓 “理性的恋爱脑”，看似矛盾但却也存在。 但生活从来都不是游戏，更不能将其情绪化，成年人的世界里理性应该永远大于感性。 喜欢村上春树的一句话 :“你要做一个不动声色的大人了，去过自己另外的生活，不是所有的鱼都会生活在同一片海里。”
